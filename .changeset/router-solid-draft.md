Add `@effector/router-solid`, a new SolidJS bindings package that mirrors the `@effector/router-react` API (`RouterProvider`, `createRouteView`, `createLazyRouteView`, `createRoutesView`, `Link`/`useLink`, `Outlet`, `withLayout`, `useRouter`, `useIsOpened`, and `useOpenedViews`) using Solid primitives, targeting SolidJS 1.x.
